## 📦 Project Modules

<div align="center">

<table style="width:90%">
  <tr>
    <th>Module</th>
    <th>Functionality</th>
  </tr>
  <tr>

  </tr>
  <tr>
    <td>🖱️ <b>Input Handler</b></td>

    <td>🔄 <b>Transformation Engine</b></td>
    <td>Handles object movement, rotation, and scaling</td>
  </tr>
  <tr>
    <td>🖼️ <b>Display Renderer</b></td>
    <td>Draws and updates scenes dynamically</td>
  </tr>
  <tr>
    <td>📝 <b>Annotation System</b></td>
    <td>Adds contextual labels and info in the simulation</td>
  </tr>

## ⚙️ How to Run the Project

> 📌 **Main File to Run**: `main.cpp`

<div align="center">

<table style="width:90%">
  <tr>
    <th>Step</th>
    <th>Instructions</th>
  </tr>
  <tr>
    <td>1️⃣</td>
    <td>Install OpenGL and FreeGLUT libraries</td>
  </tr>
  <tr>
    <td>2️⃣</td>
    <td>Clone this repository</td>
  </tr>
  <tr>
    <td>3️⃣</td>
    <td>
      Compile the project:<br>
      <b>Linux:</b><br>
      <code>g++ main.cpp -lGL -lGLU -lglut -o desktop_simulation</code><br>
      <code>./desktop_simulation</code><br><br>
      <b>Windows:</b><br>
      Create new project → Add files → Link `opengl32.lib`, `glut32.lib`
    </td>
  </tr>
  <tr>
    <td>4️⃣</td>
    <td>Run the compiled binary or hit ▶ Run in IDE</td>
  </tr>
  <tr>
    <td>5️⃣</td>
    <td>
      Controls:<br>
      • <code>Enter</code> – Start simulation<br>
      • <code>Esc</code> – Exit simulation<br>
      • Arrow keys / Mouse – Navigate & explore
    </td>
  </tr>
</table>

</div>
